# TCP Client-Server

A TCP Client-Server which enables the user to transfer files between Client and Server, as well as allowing the user to create/delete files and directories.

## Compilation

To compile:
```
make
```
To delete compiles files:
```
make clean
```

### Run Program

To Run Client:
```
./client IP-address
```
To Run Server:
```
./server
```
